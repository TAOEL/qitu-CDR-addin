## 一、设计原则

| 原则 | 说明 |
|------|------|
| **浅色柔和** | 背景 `#F8F9FA`，卡片纯白 `#FFFFFF`，配合浅色渐变卡片 |
| **三色语义** | 绿 `#81C784`（主色/成功）· 蓝 `#64B5F6`（信息）· 橙 `#FFB74D`（警告） |
| **深色点缀** | 主按钮 + 侧栏使用深海军蓝 `#1A1A2E`，形成明暗对比层次 |
| **单一真相来源** | 所有 XAML 通过 `StaticResource` 引用 DesignSystem.xaml，零硬编码 |
| **大圆角柔和** | 圆角 8/12/16/20px，传递友好现代感 |

## 七、使用规范

### 7.1 XAML 引用方式

```xml
Background="{StaticResource BgPrimaryBrush}"
Foreground="{StaticResource TextPrimaryBrush}"
FontFamily="{StaticResource PrimaryFont}"
FontSize="{StaticResource FontSizeMd}"
Margin="{StaticResource SpacingLg}"
<Button Style="{StaticResource PrimaryButtonStyle}" Content="确定" />
<TextBox Style="{StaticResource StandardTextBoxStyle}" />
```

### 7.2 禁止事项

| ❌ 禁止 | ✅ 正确 |
|--------|--------|
| `Foreground="#333333"` | `Foreground="{StaticResource TextPrimaryBrush}"` |
| `Background="#FFFFFF"` | `Background="{StaticResource BgSecondaryBrush}"` |
| `FontSize="14"` | `FontSize="{StaticResource FontSizeMd}"` |
