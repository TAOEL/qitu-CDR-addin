#### S10a 批量转曲功能详情

**文件位置**：
- ViewModel: `plugin/QiTuCDR/ViewModels/TextToCurvesViewModel.cs`
- View: `plugin/QiTuCDR/Views/TextToCurvesView.xaml`

**功能特性**：
- 三种处理范围：仅选中 / 当前页面 / 整个文档
- 三种对象类型：文本（Type=6）/ 图形（Type=1,2,4,19,21,26）/ 轮廓（Outline.Width > 0）
- 智能检测对象并统计数量
- 实时日志记录（UI 列表 + 文件日志）
- 配置参数持久化（保存到 `%LocalAppData%\QiTuCDR\TextToCurves\config.json`）
- 支持群组和 PowerClip 递归处理
- 支持操作撤销（BeginCommandGroup/EndCommandGroup）
- Toast 消息提示（成功/失败震动动画）
- COM 集合枚举（兼容 .NET IEnumerable / COM _NewEnum / dynamic 回退）

## 六、开发规范速查

| 规范 | 要求 |
|------|------|
| 开发优先级 | PC 端 → Web 端（PC 核心功能完成前不启动 Web 开发） |
| 模块开发顺序 | UI 静态页面 → 后端 API → 前后端联调 → 交互完善 |
| 迭代粒度 | 每次仅开发一个可独立验证的切片，测试跑通后才能继续 |
| 代码注释 | C# XML 注释（公开方法必须）、PHP/JS 中文注释 |
| CDR 版本 | 开发测试仅用 V26（CorelDRAW 2024），禁止 V27 |
| 文档同步 | 功能/架构有重大变更时同步更新 PRD / 架构 / 项目状态三文档 |
| 提交规范 | feat: / fix: / docs: / refactor: / test: 前缀 |
| CDR wpfhost 规范 | 禁止 Popup.AllowsTransparency + DropShadowEffect + CornerRadius（黑角 bug）；动画需用 DispatcherTimer 保证稳定 |
